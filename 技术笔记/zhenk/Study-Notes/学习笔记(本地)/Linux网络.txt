nginx：
  添加CentOS 7 Nginx yum资源库,打开终端
         sudo rpm -Uvh http://nginx.org/packages/centos/7/noarch/RPMS/nginx-release-centos-7-0.el7.ngx.noarch.rpm

  使用yum命令从Nginx源服务器中获取来安装Nginx
       sudo yum install -y nginx

 可以在任何路径下执行此命令来启动Nginx:
       sudo systemctl start nginx.service

 CentOS 7 开机启动Nginx
       sudo systemctl enable nginx.service

 显示你的服务器的公共IP地址:
       ip addr show eth0 | grep inet | awk '{ print $2; }' | sed 's/\/.*$//'

 通过此代码可以查看配置文件中哪里出现问题
       nginx -t -c /etc/nginx/nginx.conf

  启动 在安装目录下：start nginx
  停止            ：nginx.exe -s stop或 nginx.exe -s quit
 【注】stop是快速停止nginx，可能并不保存相关信息；quit是完整有序的停止nginx，并保存相关信息。
  重新载入修改后的配置 Nginx ： nginx.exe -s reload
  重新打开日志文件：            nginx.exe -s reopen
  查看Nginx版本：               nginx -v
